## 以文件流来检验输入输出而不是一行行的输入 (在提交的时候应该注释掉)

```C
// in.txt为要读取的文件
// r为只读
// stdin为标准输入
freopen("in.txt","r",stdin);  

```
- 最后提交的时候一定要注释掉，不然会有Runtime Error

## 一键给数组赋初值

```C
#include<string.h>
memset(array, 0, sizeof(array));
```
- 其中array可以是任意指针
- 0这个地方可以为任意特定的值
- 第三个地方指定长度
- 作用为：将内存的前n个字节设置为特定的值
